# Online-Learning-UI

Online-Learning-UI is a modern and responsive Online Learning Platform user interface developed using HTML, CSS, JavaScript, and Bootstrap. The project focuses on creating an engaging and user-friendly educational website design for students and instructors.

##  Features

### Learning Platform Pages

* Home Page
* Courses Page
* Course Details Page
* Categories Section
* Instructor Profiles
* Student Dashboard UI
* About Us Page
* Contact Page

###  UI Features

* Responsive Design
* Modern Layout
* Interactive Navigation Bar
* Course Cards
* Testimonials Section
* Pricing Plans
* FAQ Section
* Footer with Social Links

### 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
